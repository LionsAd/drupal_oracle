There are two ways to install Drupal 8 Oracle Database Driver:

1. Composer way. You should add one more installer-paths to yours composer.json:
"web/drivers/lib/Drupal/Driver/Database/{$name}":["type:drupal-database-driver"]
Do a standard command: `composer require drupal/oracle`.

2. Download an archive from drupal.org and place the code to this directory:
DRUPAL_ROOT/drivers/lib/Drupal/Driver/Database/oracle
